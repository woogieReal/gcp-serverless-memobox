## [2026-06-19] 프로젝트 초기 세팅

### 작업 내용
- `npm init -y` 로 `package.json` 초기화 (`main: index.js` 확인)
- `express`, `@google-cloud/storage` 의존성 설치
- 프로젝트 루트에 `index.js` 생성 — Express 앱 초기화 및 Cloud Functions 엔트리포인트 구성

### 주요 결정 사항
- **라우터 분리 구조 채택:** `index.js`에서 `/memoApi` 경로를 `routes/memo.js`로 위임하는 구조로 설계. 엔트리 파일을 얇게 유지하고 STEP 4에서 작성할 라우터 로직과 명확히 분리.
- **Cloud Functions 엔트리포인트 형식:** `exports.memoApi = app` 형태로 Express 앱을 내보내 Cloud Functions Gen 2가 인식할 수 있는 구조로 작성.

### 이슈 & 해결
- `uuid@9` deprecated 경고 발생 — `@google-cloud/storage`의 전이 의존성으로 인한 것. 직접 사용하는 패키지가 아니므로 별도 조치 없이 진행.
