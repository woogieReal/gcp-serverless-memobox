# URL 경로 중복 제거 작업계획

## 목표

현재 API URL이 `https://...cloudfunctions.net/memoApi/memoApi/` 형태로 `memoApi`가 두 번 중복되어 있다.
첫 번째는 Cloud Functions 함수 이름, 두 번째는 Express 라우터 마운트 경로(`/memoApi`)에서 비롯된 것이다.
`index.js`의 라우터 마운트 경로를 `/`로 변경해 URL을 `https://...cloudfunctions.net/memoApi/`로 단순화한다.

## 작업 범위

- `index.js` Express 라우터 마운트 경로 수정 (`/memoApi` → `/`)
- 배포 후 엔드포인트 동작 확인

---

## 세부 작업 목록

### 1단계: index.js 수정

- [X] `app.use('/memoApi', ...)` → `app.use('/', ...)` 로 변경

### 2단계: 배포 및 검증

- [X] `main` 브랜치에 커밋 & 푸시 → GitHub Actions 파이프라인 성공 확인
- [X] `GET https://...cloudfunctions.net/memoApi/` — 목록 조회 정상 응답 확인
- [X] `POST https://...cloudfunctions.net/memoApi/` — 메모 생성 정상 응답 확인
- [X] `GET https://...cloudfunctions.net/memoApi/:filename` — 내용 조회 정상 응답 확인
- [X] `PUT https://...cloudfunctions.net/memoApi/:filename` — 수정 정상 응답 확인
- [X] `DELETE https://...cloudfunctions.net/memoApi/:filename` — 삭제 정상 응답 확인
