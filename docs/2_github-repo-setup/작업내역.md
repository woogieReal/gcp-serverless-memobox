## [2026-06-19] GitHub Repository 연동

### 작업 내용
- GitHub Repository Settings에서 `Production` Environment 생성
- `Production` Environment Secrets에 `GCP_SA_KEY` 등록 (`terraform/sa-key.json` 내용 전체)

### 주요 결정 사항
- **Repository Secrets 대신 Environment Secrets 사용:** `GCP_SA_KEY`를 Repository 전체에 노출하지 않고 `Production` Environment에만 한정하여 키 노출 범위를 최소화. 이후 CI/CD 워크플로우에서 `environment: Production`을 명시해야만 해당 Secret에 접근 가능.

### 이슈 & 해결
- 특이사항 없음. 웹 콘솔에서 순서대로 진행하여 완료.

### 변경 파일
- 코드 변경 없음 (GitHub 웹 콘솔 작업)
- `docs/2_github-repo-setup/작업계획.md` (신규)
