### 주요 결정 사항
- **배포 과정에서 필요한 GCP 권한을 모두 Terraform으로 관리:** 콘솔 직접 조작 없이 `main.tf`에 추가하여 재현성 확보. 총 5개 API 활성화 및 2개 IAM 권한 추가.

### 이슈 & 해결
- **Cloud Functions API 비활성화** — Terraform `google_project_service`로 해결
- **Cloud Build / Cloud Run / Artifact Registry / Cloud Resource Manager API 비활성화** — 동일하게 Terraform으로 일괄 활성화
- **`iam.serviceaccounts.actAs` 권한 누락** — `roles/iam.serviceAccountUser`를 컴퓨트 서비스 계정에 바인딩하여 해결
- **`run.services.setIamPolicy` 권한 누락** — `roles/run.admin`을 프로젝트 레벨로 부여하여 해결
- **Node.js 20 deprecation 경고** — `google-github-actions/auth`, `setup-gcloud`를 v3로, Cloud Functions 런타임을 `nodejs22`로 업그레이드하여 해결
